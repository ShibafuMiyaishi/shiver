# Phase完了チェック

フェーズの完了条件を検証するスキル。
`/phase-check` で呼び出す。

## 手順

1. `docs/TASKS.md` を読み、現在のPhaseを特定する
2. 現在のPhaseの全タスクの状態を確認する
3. 完了条件を1つずつ検証する:
   - コードが存在するか（ファイル・関数の存在確認）
   - 必要なエンドポイントが応答するか
   - フロントエンドが起動してエラーがないか
4. 結果を一覧で報告する
5. 全条件を満たしていれば `docs/TASKS.md` のPhase状態を更新する

## Phase 1 完了条件

- カメラ顔追跡でアバターのまばたき・口開閉・眉上下・首振りが動く
- 顔未検出時も自動まばたき（3-5秒周期）が動作する
- 呼吸モーションで体が微妙に上下する

## Phase 2 完了条件

- プロンプト入力から3分以内にアバター完成
- 髪が首振りに連動して自然に揺れる
- ウィンクと照れが即時発動
- 瞳がXY方向に追従する
